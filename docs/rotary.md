# Guide to set up a rotary phone as a landline

## Hardware

- [Rotary dial phone](https://a.co/d/fRwcqi1) ~$40.
- Analog telephone adapter (ATA) [Grandstream HT801v2](https://a.co/d/1d57qn6) ~$40. Converts one analog phone line.
- Wifi router with Ethernet port (Airport Express).

## Software

- [voip.ms](https://voip.ms/en/invite/NTE5MTQy) to purchase a phone line ($1/mo). See [instructions on how to get a line](voip_ms.md).

## Installation

![IMG_1171](https://github.com/user-attachments/assets/061c2c99-c4bd-43c3-8fa6-e492367a3b43)

- Connect Rotary phone to ATA, and ATA to Wifi router.
- Once connected, dial ***02 on the phone to hear the ATA IP address.
- Configure the ATA to use the voip.ms phone line: https://wiki.voip.ms/article/Grandstream_HT802v2.
