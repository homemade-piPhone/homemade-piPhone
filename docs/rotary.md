# Guide to set up a rotary phone as a landline

## Hardware

- [Rotary dial phone](https://a.co/d/fRwcqi1) ~$40.
- Analog telephone adapter (ATA) [Grandstream HT801](https://a.co/d/1d57qn6) ~$40. Converts one analog phone line.
- Wifi router with Ethernet port (Airport Express).

## Software

- voip.ms to purchase a phone line ($1/mo). Keep voip.ms account number & password in a password manager.

## Installation

- Connect Rotary phone to ATA, and ATA to Wifi router.
- Once connected, dial ***02 on the phone to hear the ATA IP address.
- Configure the ATA to use the voip.ms phone line (https://wiki.voip.ms/article/Grandstream_HandyTone_802_-_HT802)
